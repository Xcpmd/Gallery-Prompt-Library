# <center> Gallery Prompt

![展示](./public/show.jpeg)

Gallery Prompt 是一个轻量级的提示词（Prompt）管理工具，专为 AI 绘画爱好者设计。它提供了美观的界面，帮助您分类、搜索、收藏常用的提示词，并支持正面/负面提示词的快速组合与复制。

## 功能特点

- **多级分类管理**：以 分类-子分类-提示词 为结构管理, 方便直观。
- **实时搜索**：使用 关键词搜索/模糊搜索, 速度快, 支持搜索提示词内容及翻译。
- **复制选项**：可对复制内容进行 下划线替换/括号转义。
- **实时保存**：对提示词及分类进行修改时均会触发保存操作。

## 界面预览

- **主界面**：左侧分类树，中间提示词卡片，右侧购物车。
- **导航栏**: 中间搜索框, 使用节流搜索, 右侧显示**连接状态**及**开启新世界**。
- **配色方案**：柔和毛玻璃效果，适配各色背景。
- **图标**：使用 **FontAwesome 6 free** 图标，清晰直观。
- **字体**: 英文使用**CeraRoundPro**, 中文使用 **优设鲨鱼菲特健康体**, 手写风格

## 安装与运行

### 环境要求

- Python 3.8+
  - Flask
  - python-Levenshtein (可选)

### 快速开始

1. 克隆或下载本项目。
2. 安装依赖：

   ```bash
   pip install -r ./requirements.txt
   ```

3. 启动:

   ```bash
   python app.py
   ```

   或运行 run.bat
   
   - 参数:
      - --backup  启用备份 路径: ./prompts/backUp/
      - --debug
